ENGLISH
This project retrieves weather forecast data (Temperature, Humidity %, and Wind Speed) for the next 12 hours in 12 cities from a weather API and exports it to a Redshift database.

In the root folder, you will find all the files needed to build the image using the Dockerfile.

In the Airflow_Dockerfile folder, you will find everything required to run this ETL automatically on a daily basis.


SPANISH
Este trabajo consiste en obtener una API del pronóstico del clima de las próximas 12 horas (Temperatura, % de Humedad y velocidad del viento) en 12 ciudades y exportarlo en una base de datos de Redshift. 

En la raíz de la carpeta se encuentra todo lo necesario para crear la imagen mediante el Dockerfile

Dentro de la carpeta Airflow_Dockerfile se encuentra todo lo necesario para hacer correr este ETL de manera aumática diariamente
